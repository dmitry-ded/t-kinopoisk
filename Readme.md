## Задача приложения

Веб-сервис для просмотра информации о фильмах и взаимодействия с ними. Возможность просматривать карточки фильмов, сохранять избранное, оценивать фильмы, писать комментарии и создавать собственные подборки - как на популярных кинопорталах.

---

## Целевая аудитория

- Кинолюбитель, который хочет быстро найти фильмы и смотреть описание, постер и рейтинги.
- Пользователи, которые любят оценивать проссмотренное, оставлять комментарии.
- Люди, которые создают собственные подборки фильмов на разные темы.

---

## Возможности приложения

- **Главная страница** - каталог фильмов, которые берутся с Kinopoisk API.
- **Страница фильма** - подробная информация, добавление в избранное, добавление в список, комментарии.
- **Рейтинг пользователей** - отдельная страница с фильмами, у которых есть оценки от пользователей сайта.
- **Избранное** - фильмы, которые понравились пользователю.
- **Списки фильмов** - добавление фильмов в список(с названием и описанием), публичные и приватные списки.
- **Сообщество** - возможность просматривать публичные списки пользователей.
- **Регистрация и авторизация** - JWT cookie, защищённые маршруты профиля.

## Запуск проекта локально

### Требуется

- Node.js 18+
- Java 17+
- Docker

### 1.База данных

Из папки `backend`:

```bash
docker compose up -d
```

### 2. Бэкенд

```bash
cd backend
mvn spring-boot:run
```

Сервер доступен по: `http://localhost:8080`.

### 3. Фронтенд

Нужно создать файл `frontend/.env`:

```env
VITE_POISKKINO_BASE_URL=https://api.poiskkino.dev
VITE_POISKKINO_API_KEY=ключ_api_из_тг_бота
```
В терминале:

```bash
cd frontend
npm install
npm run dev
```

Приложение доступно по: `http://localhost:5173`
